# Отчёт о тестировании Credit Card Number Validator

## Краткое описание

19.07.2021 - 23.08.2021 было проведено Классы эквивалентности, Граничные значения приложения Credit Card Number Validator.

На тестирование затрачено: 5 часов

В результате тестирования выявлены следующие дефекты:
* [issues 1](https://github.com/stodoriko/java_homework_1.1/issues/1)

## Описание процесса тестирования

В процессе тестирования использовались следующие артефакты*:
Test Case №1
Test Case №2


## Тест-кейс #1

Название: 
Ввод валидного номера банковской карты 

Приоритет: 
Высокий

Предусловие: 
код находится тут--> https://github.com/netology-code/javaqa-homeworks/tree/master/intro

Шаги для воспроизведения:
1. Скопировать код из задачи №1(Credit Card Number Validator) и вставить его в среду разработки
2. Заменить исходное значение переменной number на валидное(см.ниже)
3. Запустить код

Фактический результат:
